# 🩸 Blood Group Prediction Using Fingerprint Images and VGG16

## Project Overview
This project uses VGG16 transfer learning to classify fingerprint images into eight blood group classes.

### Features
- VGG16 Transfer Learning
- Image Preprocessing
- Data Augmentation
- Multi-class Classification

### Technologies
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Installation
```bash
pip install -r requirements.txt
jupyter notebook
```

Open `Blood_Group_VGG16_Improved.ipynb` and run all cells.
